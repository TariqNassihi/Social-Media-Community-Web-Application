# VUB Hub – Interactive Blog Platform for Students

## Overview
VUB Hub is an interactive web platform designed for students to share posts, participate in discussion threads, and interact with other users.

The platform allows users to create content, follow other users, comment on posts, react to content, and explore posts geographically using a map interface

  
## Technologies

Frontend:
- React
- JavaScript
- Bootstrap
- CSS

Backend:
- Node.js
- Express
- Passport authentication

Other tools:
- REST APIs
- Map integration for geolocation posts
